# Sound File Placeholder

**IMPORTANT:** Place your `faah.mp3` file in this directory.

The extension expects an MP3 file named `faah.mp3` in this folder.

You can use any sound effect you like - a scream, a funny sound, or the classic "FAAAAH" sound.

## Finding Sound Files

You can download free sound effects from:

- freesound.org
- zapsplat.com
- Or record your own!

See [SOUND-RESOURCES.md](../SOUND-RESOURCES.md) in the root folder for detailed instructions.

Make sure the file is named exactly: `faah.mp3`
